# Steel Brainrot - Key System

Система управления ключами для доступа к программному обеспечению.

## Функции

- ✅ Активация ключей
- ✅ Просмотр доступных ключей
- ✅ Статус использования
- ✅ Удобный интерфейс

## Использование

1. Перейдите на сайт
2. Введите полученный ключ
3. Активируйте и получите код
4. Используйте код в программе
